USESO Scavenger Hunt.

How the Hunt Works

The hunt works in the following manner. This first page is going to give you a clue, which will lead you to the next url. To guess the next page, go to will-s-h.github.io/sh/(the answer you think it is). Make sure to include the /sh, which stands for scavenger hunt! If you are correct, the website should show the next clue; if not, you will get a 404 error.

The answer will consist only of lowercase letters and numbers.

Rules

Rule 1

The only thing you’re allowed to do website-wise is test urls by going to will-s-h.github.io/(the answer you think it is). You are not allowed to look at the source code of this website in any way.

Rule 2

If you’re cheating on a scavenger hunt, that is so sad. It hasn’t stopped people before though so, if you’re caught cheating, you will be disqualified.

Rule 3

You can work in groups OR individually.

Start!
